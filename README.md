About Project:
1. Backend - WordPress + WordPress API
2. Frontend - React: 16.2.0
Token is saved in cookie after auth.

How to run:
1. Create 'credentials.js' file in '/src/'
Example:
export const adminAccess = {
    login: 'admin',
    password: '123456',
};
2. npm i
3. npm run dev

For production build:
1. npm run prod
